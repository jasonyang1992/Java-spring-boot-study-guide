# Java-spring-boot-study-guide
Sharpen your skills and prepare for interviews with this tool! It generates 20 random Java and Spring Boot questions each time you run it, covering core concepts, best practices, and practical scenarios.

### Features:

- Randomized question sets on each run
- Covers core Java and Spring Boot topics
- Ideal for interview prep or self-assessment

Get ready, practice consistently, and ace your next Java/Spring Boot interview!
